# Launch playbook

All copy below is ready-to-paste. Personalize bracketed `[name]` slots and trigger the channels in tier order. Keep all four tiers within ~4 hours of going live.

## Order
1. `show-hn.md` → submit Tue or Wed 9am ET. Stay reachable for 4 hours.
2. `awesome-pr.md` → PR to `punkpeye/awesome-mcp-servers`.
3. Submit to PulseMCP, Smithery, Glama, MCP.so, official registry — `submissions.md`.
4. Reddit: 3 posts → `reddit-*.md`.
5. Indie Hackers + Product Hunt → `indie-hackers.md`, `product-hunt.md`.
6. Tweet from `@mcpindex` and personal handle → `tweet.md`.
7. LinkedIn → `linkedin.md`.
8. Strategic single-shot DMs → `dm-anthropic.md`, `dm-mcp-builders.md`.
9. Newsletter / podcast pitches → `pitch-newsletter.md`.

Then start the **acquirer ladder** at Day 21 with `email-A-defensive.md`, `email-B-strategic.md`, `email-C-anthropic-postlaunch.md`.

## Voice rules (apply everywhere)
- No "leverage", "unlock", "powerful", "seamless", "in the age of AI", "revolutionary", "robust", "scalable", "world-class", "cutting-edge", "next-generation", "best-in-class", "synergies", "at scale".
- Human voice. No exclamation marks. No emojis.
- Specific numbers over adjectives. "3,510 servers" beats "thousands."
- Sign-off: cold emails end with `Thank you, GB`. Show HN, Reddit, Twitter, LinkedIn use the platform's native voice — no sign-off.
